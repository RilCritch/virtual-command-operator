# Project Docs

Developer-facing documentation for planning, architecture, and decisions.

## How to use this folder
- Keep user-facing docs in the repo root README.
- Keep planning, architecture, and decision records here.
- Update docs as phases progress to avoid drifting plans.

## Index

### Planning
- project-docs/plans/
  - Contains overarching plans and phase plans.
- project-docs/plans/tui-chat-implementation-plan.md
  - Overarching phased implementation plan.

### Project Idea
- project-docs/initial-project-idea.md
  - Initial review, goals, architecture summary, staged plan, references.

### Decisions
- project-docs/decisions/
  - Architecture Decision Records (ADRs).

### Architecture
- project-docs/architecture/
  - High-level architecture notes and diagrams.

### References
