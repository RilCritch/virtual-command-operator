# VCO (Virtual Command Operator)

VCO (Virtual Command Operator) is a terminal-first shell and AI chat application built with Textual and backed by the OpenCode server. Chat Mode (colloquially referred to as Virtual Chat Operator) provides a versatile AI chat interface alongside the default Shell Mode.

This repository is in the planning phase. Implementation will begin after Phase 0 planning.
